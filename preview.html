GATE Prep Question LaTeX Preview

Upload a JSON file to preview how LaTeX formatting will render in the GATE Prep app.

Supported Formats

This preview tool supports:

  • LaTeX math expressions: $E = mc^2$
  • Markdown images: ![alt text](path/to/image.png)
  • Code blocks: ```language code here ```
  • Inline code: `code`
  • Tables with | separators, including pseudocode

For questions with embedded images, make sure images are in their relative paths as specified in the markdown.
